# Tarea 1: XML

Este XML contiene información sobre dos mangas: Lady Óscar y Los Caballeros del Zodiaco. Cada manga está representado con título, autor, género, descripción, precio y stock.

Se utiliza un namespace (col:) para mantener las etiquetas bien formadas y evitar conflictos. El documento tiene un único elemento raíz <col:colecciones> para cumplir con las reglas de XML.
